# 2FA_at_medisafe_presentation_BA
Business Analyst Project – Medisafe Australia 2FA Security Enhancement  Analysed and redesigned customer authentication processes to support regulatory compliance and improve account security.

Conducted a Business Analysis project focused on improving customer account security and ensuring compliance with new privacy legislation through the implementation of Two-Factor Authentication (2FA). The project involved stakeholder interviews, requirements elicitation through questionnaires, analysis of current business processes, and identification of security and compliance gaps.

Key deliverables included documenting business requirements, developing user stories, creating current-state and future-state process flowcharts, defining the Minimum Viable Product (MVP), and presenting a solution proposal to stakeholders. The recommended solution introduced SMS-based One-Time Password (OTP) authentication, real-time security monitoring capabilities, and an enhanced customer verification process designed to improve both security and user experience while meeting an eight-week regulatory deadline.
